Funciones

Lo importante para vender sin complicarse.

La app concentra las tareas del mostrador en una experiencia directa para Android.

Punto de venta

Escaneo por cámara, búsqueda manual, carrito y cierre de venta con cliente final o cliente registrado.

Inventario editable

Productos por código de barra con nombre, precio, costo, categoría, proveedor y notas.

Productos por kilo

Ideal para pan, fiambres, frutas o cualquier producto pesado: pide el peso y calcula el precio.

Clientes y créditos

Registra clientes automáticamente, autocompleta nombres y muestra deudas pendientes.

Historial y PDF

Guarda reportes simples para revisar ventas y personas deudoras sin depender de planillas externas.

Días anteriores

Al iniciar un nuevo día, archiva el anterior para verlo, descargarlo, restaurarlo o eliminarlo.

Flujo de uso

Hecho para mostrador, no para perder tiempo configurando.

El recorrido se mantiene corto para que vender, revisar y cerrar el día sea rápido.

1. Cargar productos

Escaneas el código y guardas datos del producto.

2. Vender

Escaneas, sumas al carrito y finalizas la venta.

3. Controlar clientes

La app registra compras y deudas por nombre.

4. Cerrar día

El día queda guardado para volver si hace falta.
